# Starbase Defender

円形の宇宙域を舞台に、迫りくる敵艦隊から中央の宇宙基地を守り抜くターン制戦術シミュレーションゲームです。

[Game](index.html)

## 概要

プレイヤーは防衛艦を操作し、中央の宇宙基地を破壊しようとする敵を迎撃し、撤退させるのが目的です。
移動や攻撃にはエネルギーが必要であり、限られた資源を管理しながら戦略的に立ち回る必要があります。

## 重要事項

* このゲームの目的は、敵機を撤退させて、こちらの防衛力を示すことです。
  * 全ての敵を撤退させると、基地への攻撃はおさまり、そのステージはクリアされます。
  * もたもたしていると、増援が出現します。

* 敵を撃墜して全滅させるのが目的ではありません！
  * 撃墜してしまうと、敵意をあおってしまい、かえって状況を悪化させます。

## 操作方法

ゲームはすべて**マウスクリック（または画面タップ）**で操作します。プレイヤーが1回行動すると、敵のターンに移行します。

### 1. 移動
* **操作方法**: 宇宙空間の何もない場所をクリックします。
* **仕様**: 
  * 自機からクリックした位置に向かって移動します。
  * 1ターンに移動できる最大距離（`MAX_MOVE_DIST`）が決まっており、それを超える位置をクリックした場合は、最大距離の限界まで移動します。
  * 移動距離に応じてエネルギーを消費します。
  * 宇宙域に点在する星（`barStars`）などの障害物を突き抜ける移動はできません（ブロックされます）。

### 2. 攻撃（フェイザー砲）
* **操作方法**: 画面上の敵（緑色の機体）をクリックします。
* **仕様**:
  * 1発ごとに固定のエネルギー（`PHASER_COST`）を消費します。
  * **距離減衰**: 敵との距離が離れるほど、与えられるダメージが減少します。
  * **射程制限**: 自機の周囲に表示される2つの円（有効射程・最大射程）の外にいる敵には攻撃が届きません。
  * **射線遮蔽（LOS）**: 自機と敵の間に星（`barStars`）や宇宙基地がある場合、射線が遮られて攻撃できません。

### 3. ドッキング（補給）
* **操作方法**: 自機が中央の宇宙基地に接近した状態で、宇宙基地（黄色の円）をクリックします。
* **仕様**:
  * 宇宙基地のドッキング範囲内（`DOCKING_RANGE`）にいる場合、自機のシールドが100%まで回復し、エネルギーが補給されます。
  * 範囲外からクリックした場合はドッキングできず、ログに警告が表示されます。

## ゲームルール

### 1. ターンシステム
* プレイヤーが行動（移動、攻撃、ドッキング、または射程外への空撃ちなど）を完了すると、敵のターンになります。
* 全ての敵艦が同時に移動または攻撃を行った後、次のターンに進み、プレイヤーの手番に戻ります。

### 2. ステータス管理
* **ENERGY（エネルギー）**: 自機の行動の原動力です。尽きると移動や攻撃ができなくなります。
* **SHIELDS（自機シールド）**: 自機の耐久力です。敵の攻撃を受けると減少します。
* **BASE（基地シールド）**: 中央の宇宙基地の耐久力です。敵は基地を優先して狙ってきます。

### 3. 敵の行動AI
* 耐久力が50%以上ある敵は、射程内に基地またはプレイヤーがいれば攻撃（ピンク色のビーム）を仕掛けてきます。射程外の場合は、基地やプレイヤーに向けて接近してきます。
* 耐久力が50%未満になった敵は、生存のために宇宙域の外へと撤退を開始します。円形の境界線の外に追い出せば撃破扱いとなります。
* 敵はターン毎に一定量の耐久力を自己修復（リジェネ）します。

## 勝利・敗北条件

### 勝利条件
* 宇宙域に侵入したすべての敵（`barEnemies`）を撃破（または宇宙域外へ撤退）させる。
* 達成するとログに `MISSION ACCOMPLISHED` と表示されます。

### 敗北条件
* **宇宙基地の陥落**: 宇宙基地のシールドが 0% になる（`BASE DESTROYED`）。
* **自機の撃破**: 自機のシールドが 0% になる（`ENTERPRISE DESTROYED`）。
