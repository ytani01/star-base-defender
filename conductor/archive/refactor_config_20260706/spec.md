# Specification: オブジェクト設定変数のリファクタリング

## 概要 (Overview)
このTrackでは、現在 `OBJ_CONF.PLAYER.W` のような汎用オブジェクトに格納されているゲームの各種設定変数を、適切なクラス変数（例: `PlayerShip.W`, `EnemyShip.W`）にリファクタリングします。設定値をそれぞれのエンティティクラス内にカプセル化することで、コードの保守性と可読性を向上させることを目的としています。

## 機能要件 (Functional Requirements)
- **プレイヤー変数のリファクタリング:** `OBJ_CONF.PLAYER.*` のすべてのインスタンスを、`PlayerShip` クラスのクラス変数（例: `PlayerShip.W`）に置き換える。
- **敵変数のリファクタリング:** 敵の各種設定（`OBJ_CONF.ENEMY.*` など）を、`EnemyShip`（または関連する）クラスのクラス変数に置き換える。
- **参照箇所の更新:** コードベース全体をスキャンし、これらの設定を参照している箇所を新しいクラス変数の構文に正しく更新する。
- **動作の維持:** これは純粋なリファクタリングタスクであり、ゲームのロジックや動作、メカニクスに影響を与えてはならない。

## 範囲外 (Out of Scope)
- このタスクに直接関係しないその他の設定値（UIやゲーム全体の設定など）のリファクタリング。
- 新機能の追加や既存のゲームメカニクスの変更。
